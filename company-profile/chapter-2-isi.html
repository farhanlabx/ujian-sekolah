Tools Yang Saya Gunakan Saat Di

Koding Akademi

Scratch

Scratch adalah platform pemrograman visual berbasis blok yang dikembangkan oleh MIT Media Lab. Platform ini mempermudah pemula dalam mempelajari dasar-dasar logika pemrograman tanpa perlu khawatir dengan kesalahan sintaksis penulisan kode.

Selama kegiatan PKL di Koding Akademi, saya menggunakan Scratch untuk memandu siswa memahami alur logika berpikir terstruktur, pembuatan variabel, kondisi percabangan (if-else), serta sistem perulangan (looping) secara visual.

Construct 3

Construct 3 adalah software pembuat game berbasis HTML5 yang menggunakan konsep pemrograman berbasis kejadian (event sheets). Software ini memungkinkan pengembangan game 2D dengan cepat tanpa harus menulis kode baris demi baris.

Di Koding Akademi, kami mengimplementasikan Construct 3 untuk mengajarkan logika game yang lebih kompleks, seperti collision detection, penanganan variabel global dan lokal, struktur kontrol, serta perancangan antarmuka game (UI).

Roblox Studio

Roblox Studio adalah sebuah lingkungan pengembangan terintegrasi (IDE) yang digunakan untuk merancang dan membangun game 3D interaktif untuk platform Roblox menggunakan bahasa pemrograman Lua.

Saya menggunakan platform ini untuk membimbing siswa ke tingkat pemrograman tekstual nyata. Siswa diajarkan konsep Object-Oriented Programming (OOP), manipulasi properti Part melalui script, event handling (seperti mendeteksi sentuhan), serta pemrograman jaringan multipemain sederhana.
